# Twitch Website for [AlexSpinks](https://alexspinks.dev)

This project is a custom website built for my Twitch account, designed to showcase my streams, content, and offer viewers an interactive hub. The site is hosted on DigitalOcean at [alexspinks.dev](https://alexspinks.dev).

## Features

- Personal homepage for my Twitch channel.
- Highlights, schedules, and stream information.
- Easy-to-navigate design for new and returning viewers.

## Tech Stack

- **HTML:** Primary language for website structure and content.
- **DigitalOcean:** Hosting provider for fast, reliable access.

## Getting Started

To contribute or run locally:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/AlexSpinks/Twitch-website.git
   ```
2. **Open the project folder** and view `index.html` in your browser.
